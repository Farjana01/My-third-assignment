Frequently Asked Question

Layout is important when it comes to webpages. It is a long established fact that a reader will be distracted
by the readable content of a page when looking at its layout.

CSS Grid Layout is a two-dimensional grid-based layout system, designed for laying out pages and user interfaces. Flexbox is a one-dimensional layout system, designed for creating flexible boxes to display content on web pages. There are some key differences between the two systems: Grid can be used to create both horizontal and vertical layouts, while Flexbox is limited to creating vertical layouts.

Bootstrap and Tailwind are two CSS frameworks that are often compared to each other. Bootstrap was created first, and is more widely used. Tailwind is newer, and has been gaining in popularity. Both frameworks have their pros and cons, but which one is the best for you depends on your needs. Bootstrap is a well-established framework that is widely used. It has a large community of developers who are constantly creating new components and plugins.

CSS box model is a way of representing elements on a web page. It allows you to control the layout of your web page using CSS. The box model is made up of five boxes: the content box, the padding box, the border box, the margin box, and the browser window.

The HTML semantics refers to the tags that provide meaning to an HTML page rather than just presentation. It makes HTML more comprehensible by better defining the different sections and layout of web pages.
Examples of semantic elements : <header> <nav> <article> <main> <footer>
